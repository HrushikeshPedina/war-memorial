
# War Memorial – GitHub Pages Package

## How to use
1. Upload all files to a GitHub repository named `war-memorial`
2. Put your MP3 files into the /audio folder
3. Enable GitHub Pages (Settings → Pages → main / root)

## Audio naming
hero01-hi.mp3 / hero01-en.mp3
...
hero21-hi.mp3 / hero21-en.mp3
intro-hi.mp3 / intro-en.mp3

## URL format
https://HrushikeshPedina.github.io/war-memorial/hero01.html
